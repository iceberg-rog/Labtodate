## Hot path: critical safety / financial integrity (BUG-001/002/009/010/011/012/019)

### src/app/admin/actions.ts
- `bulkMarkAllShipped`: address-complete pre-filter + atomic `updateMany`
  with status precondition (kills double-fire; surfaces skip count).
- `setOrderFulfillment`: address-complete guard for SHIPPED/DELIVERED +
  idempotency pre-check (`statusUnchanged && carrierUnchanged &&
  trackingUnchanged`) + atomic `updateMany`. **This is the BUG-009
  + BUG-010 fix combined** — kills the `order.fulfillment ×17`
  duplicate-audit bug and the address-less SHIPPED path.
- `markOrderPaidManually`: proof file is now **mandatory** for
  `BANK_TRANSFER` and `RECEIPT` methods. Atomic `updateMany` with
  `status: 'PENDING_PAYMENT'` precondition prevents double-PAID.
- `verifyPayment`: race-protection via `updateMany` with
  `AWAITING_VERIFICATION + PENDING_PAYMENT` precondition. Verifies the
  count===1; the loser-of-race gets a friendly message instead of a
  duplicate audit/notification/email.
- Added `shippingAddressIsComplete()` helper used by both single +
  bulk fulfillment paths.

### src/app/api/stripe/webhook/route.ts
- `checkout.session.completed` is now **idempotent**: pre-check
  current `status === 'PENDING_PAYMENT'` + atomic `updateMany` with
  the same precondition. Stripe webhook redeliveries no longer cause
  duplicate invoice emails, duplicate notifyAdmins, duplicate
  in-app notifications, or `paidAt` drift.
- Currency is formatted via `Intl.NumberFormat(currency)` instead of
  hardcoded `€`.

### src/app/app/seller/products/actions.ts
- `publishProduct`: sellers can only move DRAFT → PENDING_REVIEW
  (admin must approve) or PUBLISHED → DRAFT (un-list). They CANNOT
  push to PUBLISHED directly. ADMIN keeps full transitions.
- `deleteProduct`: if any `OrderItem` references the product, route
  to `ARCHIVED` status instead of hard-delete. Preserves order
  history + buyer reviews + wishlist for products that ever sold.

### src/lib/cart/actions.ts
- Old `checkoutCart()` is now a redirect to `/checkout/cart`.
- New `startCartCheckoutWithAddress(formData)`: address collection
  BEFORE order creation (closes the source of address-less orders
  like Q6BEQM, 5LX2KB, TATSIL).
- Stripe-on path preserved as `_legacyStripeCartHandoff()` for future
  enablement. Currency-aware notification text.

### src/app/checkout/cart/page.tsx  *(new file)*
- Mirror of `/checkout/[slug]`: shipping form + order summary +
  manual-payment messaging (no Stripe wording while in manual mode).

### src/app/admin/page.tsx

  - the tile `<p>{sub}</p>` that may contain "Xm ago" relative time
